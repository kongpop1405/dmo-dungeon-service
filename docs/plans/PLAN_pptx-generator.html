01 Goal

  • มี gen script (node build_pptx.mjs) → regen pptx ได้ทุกเมื่อ
  • แก้ข้อมูลที่เดียว (data/poster.json) → reflect ทั้ง 4 slide
  • รองรับ: แก้ราคา/จำนวน/text · ใส่รูป boss จริง · ปรับสี/ธีม · เพิ่ม/ลบ slide (slide = element ใน array)

04 Build steps

  1. extract data จาก slide XML → เขียน data/poster.json (4 dungeon ครบ)
  2. npm init -y + npm i pptxgenjs (local node_modules — gitignore)
  3. เขียน build_pptx.mjs: loop poster.json → addSlide → วาด text/shape/รูป ต่อ zone (§03)
  4. map theme→accent color · ฝัง boss image (มี=ใส่จริง, ไม่มี=กล่อง placeholder + ชื่อ)
  5. node build_pptx.mjs → เปิด pptx ใน PowerPoint เช็คตา
